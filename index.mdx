<div className="mt-16 mb-16 max-w-5xl mx-auto px-6">
  <h2 className="text-3xl font-bold text-gray-900 dark:text-white mb-4">Quick Start</h2>
  <p className="text-lg text-gray-600 dark:text-gray-400 mb-8">Get Walrus running in minutes</p>

  <Steps>
    <Step title="Install from crates.io">
      Add Walrus to your Rust project:
      
      ```toml Cargo.toml
      [dependencies]
      walrus-rust = "0.2.0"
      ```
    </Step>
    
    <Step title="Create a WAL instance">
      Initialize Walrus and write your first entry:
      
      ```rust
      use walrus_rust::Walrus;

      let wal = Walrus::new()?;
      wal.append_for_topic("my-topic", b"Hello, Walrus!")?;
      ```
    </Step>
    
    <Step title="Read from the log">
      Consume entries from your topic:
      
      ```rust
      if let Some(entry) = wal.read_next("my-topic", true)? {
          println!("Read: {:?}", String::from_utf8_lossy(&entry.data));
      }
      ```
    </Step>
  </Steps>
</div>
